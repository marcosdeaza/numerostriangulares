EJERCICIO: NÚMEROS TRIANGULARES

DESCRIPCIÓN:
El programa pide un número y verifica si es un "número triangular".
Un número triangular es aquel que se puede formar sumando números consecutivos (1 + 2 + 3...).

DETALLES:
- El código en C++ incluye comentarios explicando la lógica.
- Se basa en un bucle que acumula sumas hasta alcanzar el número introducido.
